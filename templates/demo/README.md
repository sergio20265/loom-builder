# Demo Templates

Import these JSON files from Loom > Templates using the Import template form.

- landing-saas.json - full landing page for SaaS, services, or product launches.
- agency-home.json - agency/consulting homepage.
- portfolio-home.json - personal portfolio homepage.
- shop-home-woocommerce.json - WooCommerce-oriented shop homepage. Activate WooCommerce before importing so the product widgets are preserved by the layout sanitizer.
- header-clean.json - reusable header template with entire-site display condition.
- footer-simple.json - reusable footer template with entire-site display condition.

All layouts use Loom widgets and are designed to sit on the clean Loom Canvas theme without extra assets or a build step.
